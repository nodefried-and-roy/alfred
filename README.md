## description
an all purpose node.js bot with built in console + prompt

## operation
### compatibility
windows, osx, linux
### requirements
python 2.7, node.js 8+, npm
### installation
1) `git clone https://github.com/nodefried-and-roy/alfred.git`
2) `cd alfred; npm install -g;`
3) `node alfred.js`
